
## 简介

中文逆文本正则化 (ITN, Inverse Text Normalization) ，即将文本中的中文数字转为阿拉伯数字。

基于正则表达式实现。

## 用法示例

```python
from chinese_itn import chinese_to_num

text = '你生日那天我会送你999朵玫瑰'
result = chinese_to_num(text)

print(result)
```
